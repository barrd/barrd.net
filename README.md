# Source Code for barrd.net

You are welcome to view, clone or download this source code.

## Install Instructions

```console
npm init
npm install
```

## Gulp Commands

To compile SCSS and reload browser

```console
gulp watch
```

To compile CSS for distribution inc. autoprefixing

```console
gulp css
```
